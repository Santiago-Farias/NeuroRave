# NeuroRave

A basic electronic music manager, where you will be able to enter songs with generes like breakcore, dubstep, vocaloid, digicore, 
and wherever else.

## Technologies

- Java 21+
- Netbeans IDE
- OOP

## Next features
- Interactive console menu
- Song system with differents generes
- Simulated playbacks of songs with unique styllitc behaviors by style
